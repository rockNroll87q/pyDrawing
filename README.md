### How to use it

* Run with `python ./mouse_demo.py` or IDE;
* Use the mouse as a brush to draw what you like;
* Press `Esc` o `q` to finish.


### What you need to run it?

* Python2.x (3.x should work as well) 
* Psychopy ([link](http://www.psychopy.org/))
* a mouse :D
